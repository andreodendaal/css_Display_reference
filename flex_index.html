flexbox - Flexible Box Layout

There are two important components to a flexbox layout: flex containers and flex items.

A flex container is an element on a page that contains flex items. All direct child elements of a flex container are flex items. This distinction is important because some of the flex properties apply to flex containers while others apply to flex items.

flexbox properties:

  1. justify-content
  2. align-items
  3. flex-grow
  4. flex-shrink
  5. flex-basis
  6. flex
  7. flex-wrap
  8. align-content
  9. flex-direction
  10. flex-flow
