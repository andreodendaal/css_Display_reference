flex-wrap

In the situation where we don’t want our content to shrink to fit its container. We can make the flex items move to the next line. This can be declared with the flex-wrap property. The flex-wrap property can accept three values:

  1. wrap — child elements of a flex container will move down to the next line starting from the final item and working towards the first.
  2. nowrap — prevents items from wrapping; this is the default value and is only necessary to override a wrap value set by a different CSS rule.
  3. wrap-reverse — the wrapped element is displayed on top of the other elements in the flex container starting from the last and working toward the first (a mirror image of the wrap value).
