flex

The flex property provides a convenient way for specifying how elements stretch and shrink, while simplifying the CSS required. The flex property allows you to declare flex-grow, flex-shrink, and flex-basis all in one line. The second and third parameters (flex-shrink and flex-basis) are optional. Default is 0 1 auto. Order is: grow shrink basis

Note: The flex property is different from the flex value used for the display property.

Top

flex: 2 2 100px;

flex: 1 3 50px;

flex: 2 2 100px;

Bottom

flex: 1 2 75px;
