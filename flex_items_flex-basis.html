flex-basis

The dimensions of divs can be determined by heights and widths set with CSS. Another way of specifying the width of a flex item is with the flex-basis property. flex-basis allows us to specify the width of an item before it stretches or shrinks.

Grows

flex-basis: 60px;

flex-grow: 1;

flex-grow: 3;

flex-basis: 60px;

flex-grow: 1;

Shrinks

flex-basis: 300px;

flex-shrink: 3;

flex-basis: 150px;

flex-shrink: 2;

flex-basis: 300px;

flex-shrink: 3;
